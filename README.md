# Gym-Assistant
*Work in progress*
## Developing

### Quick start

`yarn`

`yarn android`
`yarn ios`

### Cleaning your project
Sometimes we face some issues on adding native (Android/iOS) packages. This command helps you to clean your native data:

Cleaning all platforms:
`yarn clean`


`yarn clean:ios`
`yarn clean:android`

### Linking a new package
`yarn add package`
`yarn run link:ios`

## Testing
`yarn test`
